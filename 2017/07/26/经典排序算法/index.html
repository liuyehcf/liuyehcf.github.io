6 计数排序

计数排序复杂度是O(n)，以牺牲空间复杂度来提高时间效率。适用于可枚举且范围较小的类型的排序，例如short、char、byte等

7 总结

排序算法 是否稳定 是否原址 复杂度
冒泡排序 是 是 O(N2)
插入排序 是 是 O(N2)
归并排序 是 否 O(NlgN)
堆排序 否 是 O(NlgN)
快速排序 否 是 O(NlgN)
计数排序 是(对于基本类型) 否 O(N)

可以用leetcode第912题来验证算法是否正确
