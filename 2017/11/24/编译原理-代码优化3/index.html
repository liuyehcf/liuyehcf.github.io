зј–иҜ‘еҺҹзҗҶ-д»Јз ҒдјҳеҢ–3

еҸ‘иЎЁдәҺ 2017-11-24 жӣҙж–°дәҺ 2020-12-28 еҲҶзұ»дәҺ Compile Principle йҳ…иҜ»ж¬Ўж•°пјҡ

йҳ…иҜ»жӣҙеӨҡ

1 жҙ»и·ғеҸҳйҮҸеҲҶжһҗ

жҙ»и·ғеҸҳйҮҸ

  • еҜ№дәҺеҸҳйҮҸ$x$е’ҢзЁӢеәҸзӮ№$p$пјҢеҰӮжһңеңЁжөҒеӣҫдёӯжІҝзқҖд»Һ$p$ејҖе§Ӣзҡ„жҹҗжқЎ__и·Ҝеҫ„дјҡеј•з”ЁеҸҳйҮҸ$x$еңЁ$p$зӮ№зҡ„еҖјпјҢеҲҷз§°еҸҳйҮҸ$x$еңЁзӮ№$p$жҳҜ__жҙ»и·ғ(live)__зҡ„пјҢеҗҰеҲҷз§°еҸҳйҮҸ$x$еңЁзӮ№$p$__дёҚжҙ»и·ғ(dead)

1.1 жҙ»и·ғеҸҳйҮҸдҝЎжҒҜзҡ„дё»иҰҒз”ЁйҖ”

еҲ йҷӨж— з”ЁиөӢеҖј

  • __ж— з”ЁиөӢеҖј__пјҡеҰӮжһң$x$еңЁзӮ№$p$зҡ„е®ҡеҖјеңЁеҹәжң¬еқ—еҶ…жүҖжңүеҗҺз»§зӮ№йғҪ__дёҚиў«еј•з”Ё__пјҢдё”$x$еңЁеҹәжң¬еқ—еҮәеҸЈд№ӢеҗҺеҸҲжҳҜ__дёҚжҙ»и·ғзҡ„__пјҢйӮЈд№Ҳ$x$еңЁзӮ№$p$зҡ„е®ҡеҖје°ұжҳҜж— з”Ёзҡ„

дёәеҹәжң¬еқ—еҲҶй…ҚеҜ„еӯҳеҷЁ

  • еҰӮжһң__жүҖжңүеҜ„еӯҳеҷЁйғҪиў«еҚ з”Ё__пјҢе№¶дё”иҝҳйңҖиҰҒз”іиҜ·дёҖдёӘеҜ„еӯҳеҷЁпјҢеҲҷеә”иҜҘиҖғиҷ‘дҪҝз”Ёе·Із»Ҹеӯҳж”ҫдәҶжӯ»дәЎеҖјзҡ„еҜ„еӯҳеҷЁпјҢеӣ дёәиҝҷдёӘеҖјдёҚйңҖиҰҒдҝқеӯҳеҲ°еҶ…еӯҳ
  • еҰӮжһңдёҖдёӘеҖјеңЁеҹәжң¬еқ—з»“е°ҫеӨ„жҳҜжӯ»зҡ„__е°ұ__дёҚеҝ…еңЁз»“е°ҫеӨ„дҝқеӯҳиҝҷдёӘеҖј

1.2 жҙ»и·ғеҸҳйҮҸзҡ„дј йҖ’еҮҪж•°

жҙ»и·ғеҸҳйҮҸжҳҜдёҖдёӘе…ёеһӢзҡ„йҖҶеҗ‘ж•°жҚ®жөҒй—®йўҳ

$$IN[B] = f_B(OUT[B]$$

е…¶дёӯпјҢ$f_B(x) = use_B \cup (x-def_B)$

  • $def_B$пјҡеңЁеҹәжң¬еқ—Bдёӯе®ҡеҖјпјҢдҪҶжҳҜ__е®ҡеҖјеүҚ__еңЁBдёӯ__жІЎжңүиў«еј•з”Ё__зҡ„еҸҳйҮҸзҡ„йӣҶеҗҲ
  • $use_B$пјҡеңЁеҹәжң¬еқ—Bдёӯеј•з”ЁпјҢдҪҶжҳҜ__еј•з”ЁеүҚ__еңЁBдёӯ__жІЎжңүиў«е®ҡеҖј__зҡ„еҸҳйҮҸйӣҶеҗҲ

1.3 жҙ»и·ғеҸҳйҮҸж•°жҚ®жөҒж–№зЁӢ

з¬ҰеҸ·еҸҠе…¶иҜҙжҳҺ

  1. $IN[B]$пјҡеңЁеҹәжң¬еқ—Bзҡ„е…ҘеҸЈеӨ„зҡ„жҙ»и·ғеҸҳйҮҸйӣҶеҗҲ
  2. $OUT[B]$пјҡеңЁеҹәжң¬еқ—Bзҡ„еҮәеҸЈеӨ„зҡ„жҙ»и·ғеҸҳйҮҸйӣҶеҗҲ

ж–№зЁӢ

  • $IN[EXIT] = \Phi$
  • $IN[B] = f_B(OUT[B]пјү ( B \ne EXIT )$
  • $f_B(x) = use_B \cup (x-def_B)$
  • $OUT[B]= \cup_{SжҳҜBзҡ„дёҖдёӘеҗҺз»§}\;\;\;\;\;\;\;\;IN[S], ( B \ne EXIT )$

1.4 и®Ўз®—жҙ»и·ғеҸҳйҮҸзҡ„иҝӯд»Јз®—жі•

иҫ“е…ҘпјҡжөҒеӣҫGпјҢе…¶дёӯжҜҸдёӘеҹәжң¬еқ—Bзҡ„$use_B$е’Ң$def_B$йғҪе·Іи®Ўз®—еҮәжқҘ

иҫ“еҮәпјҡ$IN[B]$е’Ң$OUT[B]$

ж–№жі•пјҡ

дҫӢеӯҗ

1.5 е®ҡеҖј-еј•з”Ёй“ҫ(Definition-Use Chains)

е®ҡеҖј-еј•з”Ёй“ҫпјҡи®ҫеҸҳйҮҸ$x$жңүдёҖдёӘе®ҡеҖј$d$пјҢиҜҘе®ҡеҖјжүҖжңүиғҪеӨҹеҲ°иҫҫзҡ„еј•з”Ё$u$зҡ„йӣҶеҗҲз§°дёә$x$еңЁ$d$еӨ„зҡ„__е®ҡеҖј-еј•з”Ёй“ҫ__пјҢз®Җз§°$du$й“ҫ

еҰӮжһңеңЁжұӮи§Ј__жҙ»и·ғеҸҳйҮҸ__ж•°жҚ®жөҒж–№зЁӢдёӯзҡ„$OUT[B]$ж—¶пјҢе°Ҷ$OUT[B]$иЎЁзӨәжҲҗ__д»ҺBзҡ„жң«е°ҫеӨ„иғҪеӨҹеҲ°иҫҫзҡ„еј•з”Ёзҡ„йӣҶеҗҲ__пјҢйӮЈд№ҲпјҢеҸҜд»ҘзӣҙжҺҘеҲ©з”ЁиҝҷдәӣдҝЎжҒҜи®Ўз®—еҹәжң¬еқ—BдёӯжҜҸдёӘеҸҳйҮҸ$x$еңЁе…¶е®ҡеҖјеӨ„зҡ„$du$й“ҫ

  • еҰӮжһңBдёӯ$x$зҡ„е®ҡеҖј$d$д№ӢеҗҺжңү$x$зҡ„з¬¬дёҖдёӘе®ҡеҖј$d^{\prime}$пјҢеҲҷ$d$е’Ң$d^{\prime}$д№Ӣй—ҙ$x$зҡ„жүҖжңүеј•з”Ёжһ„жҲҗ$d$зҡ„$du$й“ҫ
  • еҰӮжһңBдёӯ$x$зҡ„е®ҡеҖј$d$д№ӢеҗҺжІЎжңү$x$зҡ„ж–°зҡ„е®ҡеҖјпјҢеҲҷBдёӯ$d$д№ӢеҗҺ$x$зҡ„жүҖжңүеј•з”Ёд»ҘеҸҠ$OUT[B]$дёӯ$x$зҡ„жүҖжңүеј•з”Ёжһ„жҲҗ$d$зҡ„$du$й“ҫ

2 еҸҜз”ЁиЎЁиҫҫејҸеҲҶжһҗ

еҸҜз”ЁиЎЁиҫҫејҸ

  • еҰӮжһңд»ҺжөҒеӣҫзҡ„__йҰ–иҠӮзӮ№__еҲ°иҫҫзЁӢеәҸзӮ№$p$зҡ„__жҜҸжқЎи·Ҝеҫ„__йғҪеҜ№иЎЁиҫҫејҸ$x\;op\;y$иҝӣиЎҢи®Ўз®—пјҢе№¶дё”д»ҺжңҖеҗҺдёҖдёӘиҝҷж ·зҡ„и®Ўз®—еҲ°зӮ№$p$д№Ӣй—ҙ__жІЎжңүеҶҚж¬ЎеҜ№$x$жҲ–$y$е®ҡеҖј__пјҢйӮЈд№ҲиЎЁиҫҫејҸ$x\;op\;y$еңЁзӮ№$p$жҳҜеҸҜз”Ёзҡ„(available)

иЎЁиҫҫејҸеҸҜз”Ёзҡ„зӣҙи§Ӯж„Ҹд№ү

  • еңЁзӮ№pдёҠпјҢx op yе·Із»ҸеңЁ__д№ӢеүҚиў«и®Ўз®—иҝҮ__пјҢдёҚйңҖиҰҒйҮҚж–°и®Ўз®—

2.1 еҸҜз”ЁиЎЁиҫҫејҸдҝЎжҒҜзҡ„дё»иҰҒз”ЁйҖ”

  1. ж¶ҲйҷӨе…ЁеұҖе…¬е…ұеӯҗиЎЁиҫҫејҸ
  2. иҝӣиЎҢеӨҚеҲ¶дј ж’ӯ
    • еңЁ$x$зҡ„еј•з”ЁзӮ№$u$еҸҜд»Ҙз”Ё$y$д»Јжӣҝ$x$зҡ„жқЎд»¶пјҡд»ҺжөҒеӣҫзҡ„__йҰ–иҠӮзӮ№__еҲ°иҫҫuзҡ„__жҜҸжқЎи·Ҝеҫ„__йғҪеӯҳеңЁеӨҚеҲ¶иҜӯеҸҘ$x = y$пјҢе№¶дё”д»ҺжңҖеҗҺдёҖжқЎеӨҚеҲ¶иҜӯеҸҘ$x = y$еҲ°зӮ№$u$д№Ӣй—ҙ__жІЎжңүеҶҚж¬ЎеҜ№$x$жҲ–$y$е®ҡеҖј__пјҲе°ұжҳҜеҸҜз”ЁиЎЁиҫҫејҸзҡ„е®ҡд№үпјү

2.2 еҸҜз”ЁиЎЁиҫҫејҸзҡ„дј йҖ’еҮҪж•°

еҜ№дәҺеҸҜз”ЁиЎЁиҫҫејҸж•°жҚ®жөҒжЁЎејҸиҖҢиЁҖпјҢеҰӮжһңеҹәжң¬еқ—BеҜ№$x$жҲ–иҖ…$y$иҝӣиЎҢдәҶ(жҲ–еҸҜиғҪиҝӣиЎҢ)е®ҡеҖјпјҢдё”д»ҘеҗҺжІЎжңүйҮҚж–°и®Ўз®—$x\;op\;y$пјҢеҲҷз§°B__жқҖжӯ»иЎЁиҫҫејҸ$x\;op\;y$гҖӮеҰӮжһңеҹәжң¬еқ—BеҜ№$x\;op\;y$иҝӣиЎҢи®Ўз®—пјҢе№¶дё”д№ӢеҗҺ__жІЎжңүйҮҚж–°е®ҡеҖј$x$жҲ–$y$пјҢеҲҷз§°B__з”ҹжҲҗиЎЁиҫҫејҸ__$x\;op\;y$

дј йҖ’еҮҪж•°е®ҡд№үеҰӮдёӢпјҡ

  • $f_B(x)= e\_gen_B вҲӘ(x- e\_kill_B)$
  • $e\_gen_B$ пјҡеҹәжң¬еқ—BжүҖз”ҹжҲҗзҡ„еҸҜз”ЁиЎЁиҫҫејҸзҡ„йӣҶеҗҲ
  • $e\_kill_B$ пјҡеҹәжң¬еқ—BжүҖжқҖжӯ»зҡ„Uдёӯзҡ„еҸҜз”ЁиЎЁиҫҫејҸзҡ„йӣҶеҗҲ
  • $U$пјҡжүҖжңүеҮәзҺ°еңЁзЁӢеәҸдёӯдёҖдёӘжҲ–еӨҡдёӘиҜӯеҸҘзҡ„еҸійғЁзҡ„иЎЁиҫҫејҸзҡ„е…ЁйӣҶ

2.3 $e\_gen_B$зҡ„и®Ўз®—

еҲқе§ӢеҢ–пјҡ$e\_gen_B = \Phi$

йЎәеәҸжү«жҸҸеҹәжң¬еқ—зҡ„жҜҸдёӘиҜӯеҸҘпјҡ$z = x\;op\;y$

  1. жҠҠ$x\;op\;y$еҠ е…Ҙ$e\_gen_B$
  2. д»Һ$e\_gen_B$дёӯеҲ йҷӨе’Ң$z$зӣёе…ізҡ„иЎЁиҫҫејҸ
  • дёҠиҝ°дёӨдёӘж“ҚдҪңйЎәеәҸдёҚиғҪйў еҖ’

2.4 $e\_kill_B$зҡ„и®Ўз®—

еҲқе§ӢеҢ–пјҡ$e\_kill_B = \Phi$

йЎәеәҸжү«жҸҸеҹәжң¬еқ—зҡ„жҜҸдёӘиҜӯеҸҘпјҡ$z = x\;op\;y$

  1. д»Һ$e\_kill_B$дёӯеҲ йҷӨиЎЁиҫҫејҸ$x\;op\;y$
  2. жҠҠжүҖжңүе’Ң$z$зӣёе…ізҡ„иЎЁиҫҫејҸеҠ е…ҘеҲ°$e\_kill_B$дёӯ

2.5 еҸҜз”ЁиЎЁиҫҫејҸзҡ„ж•°жҚ®жөҒж–№зЁӢ

з¬ҰеҸ·еҸҠе…¶иҜҙжҳҺ

  1. $IN[B]$пјҡеңЁBзҡ„е…ҘеҸЈеӨ„еҸҜз”Ёзҡ„$U$дёӯзҡ„иЎЁиҫҫејҸйӣҶеҗҲ
  2. $OUT[B]$пјҡеңЁBзҡ„еҮәеҸЈеӨ„еҸҜз”Ёзҡ„$U$дёӯзҡ„иЎЁиҫҫејҸйӣҶеҗҲ

ж–№зЁӢ

  • $OUT[ENTRY]= \Phi$
  • $OUT[B]=f_B(IN[B]), ( B \ne ENTRY )$
    • $f_B(x)= e\_gen_B \cup (x- e\_kill_B)$
  • $IN[B]= \cap_{PжҳҜBзҡ„дёҖдёӘеүҚй©ұ}\;\;\;\;\;\;\;OUT[P], ( B \ne ENTRY )$

2.6 и®Ўз®—еҸҜз”ЁиЎЁиҫҫејҸзҡ„иҝӯд»Јз®—жі•

иҫ“е…ҘпјҡжөҒеӣҫGпјҢе…¶дёӯжҜҸдёӘеҹәжң¬еқ—Bзҡ„$e\_gen_B$е’Ң$e\_kill_B$йғҪе·Іи®Ўз®—еҮәжқҘ

иҫ“еҮәпјҡ$IN[B]$е’Ң$OUT[B]$

ж–№жі•:

2.6.1 дёәд»Җд№Ҳе°Ҷ$OUT[B]$йӣҶеҗҲеҲқе§ӢеҢ–дёә$U$

е°ҶOUTйӣҶеҗҲеҲқе§ӢеҢ–дёә$\Phi$еұҖйҷҗжҖ§еӨӘеӨ§

еҰӮжһң$OUT[B_2]^0 = \Phi$

  • йӮЈд№Ҳ$IN[B_2]^1= OUT[B_1]^1 \cap OUT[B_2]^0 = \Phi$

еҰӮжһң$OUT[B_2]^0 = U$

  • йӮЈд№Ҳ$IN[B_2]^1= OUT[B_1]^1 \cap OUT[B_2]^0 = OUT[B_1]$
