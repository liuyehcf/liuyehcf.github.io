注解含义解释

  1. Warmup：配置预热参数
    • iterations: 预测执行次数
    • time：每次执行的时间
    • timeUnit：每次执行的时间单位
  2. Measurement：配置测试参数
    • iterations: 测试执行次数
    • time：每次执行的时间
    • timeUnit：每次执行的时间单位
  3. Fork：总共运行几轮
  4. BenchmarkMode：衡量的指标，包括平均值，峰值等
  5. OutputTimeUnit：输出结果的时间单位
